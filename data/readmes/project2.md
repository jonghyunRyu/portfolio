# 다무너 - AI 상담 요약 통신 플랫폼

## 🔗 배포 URL

https://da-moono-frontend-deploy.vercel.app

## 📝 프로젝트 요약

실시간 상담과 AI 기반 상담 요약을 결합한 **통합 요금제 상담 플랫폼**입니다.
7인 팀(프론트 6, 백엔드 1)에서 마이페이지, 이용 안내, 404 페이지, 데이터 시각화를 담당했습니다.

- 역할: Frontend 개발
- 기술 스택: React, TypeScript, Vanilla-Extract, Three.js, Framer Motion, Chart.js

---

## 🧩 주요 기능

- AI 챗봇 → 실시간 상담 → 상담 요약까지 **하나의 흐름으로 통합**
- 상담 이력 및 이용 데이터 **시각화**
- Three.js 기반 **3D 로딩 UI**
- 사용자 역할(ADMIN/USER)에 따른 **동적 리다이렉트 404 페이지**

---

## 🎯 문제 해결

| 문제 | 해결 방식 |
|------|----------|
| 배포 환경에서 이미지 경로가 깨지는 문제 | 상대 경로에서 import 기반 정적 자산 참조로 변경, Vite 빌드 파이프라인 활용 |
| SPA에서 잘못된 경로 접근 시 빈 화면 표시 | 와일드카드 라우팅 + 역할 기반 동적 리다이렉트 404 페이지 구현 |
| 상담 데이터 로딩 시 사용자 이탈 우려 | Three.js 3D 오브젝트 로딩 애니메이션 적용으로 체감 대기 시간 감소 |
