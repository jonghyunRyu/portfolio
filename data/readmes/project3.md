# 미니 OTT 웹 플랫폼

## 📝 프로젝트 요약

OTT 스트리밍 서비스를 모티브로 **콘텐츠 탐색 및 회원 인증 기능을 제공하는 풀스택 웹 플랫폼**입니다.

- 기간: 2025.10.13 ~ 2025.10.27
- 역할: Full Stack (개인 프로젝트)
- 기술 스택: HTML, CSS, JavaScript, Java, Spring Boot, MyBatis, MySQL

---

## 🧩 주요 기능

- 아이디/이메일 중복 검사 + 프론트·백엔드 **유효성 검증**
- 세션 기반 **로그인/로그아웃 인증 시스템**
- SHA-256 + Salt **비밀번호 해시 암호화**
- 카테고리 탭 필터링 + 정렬 기능이 있는 **콘텐츠 목록**
- 트레일러 자동재생이 포함된 **콘텐츠 상세 페이지**

---

## 🎯 문제 해결

| 문제 | 해결 방식 |
|------|----------|
| 평문 비밀번호 저장의 보안 취약점 | SHA-256 + 사용자별 Salt 값을 적용한 해시 암호화 구현 |
| 프론트엔드만으로는 입력 검증이 불완전 | 프론트엔드 + 백엔드 양쪽에서 유효성 검증 로직 적용 |
| 세션 만료 시 사용자 상태 불일치 | /check API를 통한 세션 유효성 검증 후 UI 상태 동기화 |

---

## REST API

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/signup` | 회원가입 |
| POST | `/login` | 로그인 |
| POST | `/logout` | 로그아웃 |
| GET | `/check` | 세션 유효성 확인 |
| GET | `/getContents` | 콘텐츠 목록 조회 |
| GET | `/contents/{id}` | 콘텐츠 상세 조회 |
