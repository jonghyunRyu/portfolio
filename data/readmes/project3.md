# 🎬 Mini OTT Web Platform

OTT 스트리밍 서비스를 모티브로 **콘텐츠 탐색 및 회원 인증 기능을 제공하는 웹 플랫폼**을 개발한 풀스택 프로젝트입니다.

## 📌 프로젝트 개요

- **기간**: 2025.10.13 ~ 2025.10.27
- **역할**: Full Stack
- **기술 스택**:  
  HTML, CSS, JavaScript, Java, Spring Boot, MyBatis, MySQL

---

## 🚀 주요 구현 기능

### 1. 회원가입 및 로그인 기능

- 아이디 / 이메일 **중복 검사 기능 구현**
- 프론트엔드와 백엔드에서 **유효성 검증 로직 적용**
- 사용자 입력 데이터 검증을 통해 **잘못된 요청 방지**

---

### 2. 세션 기반 사용자 인증 시스템

- 로그인 성공 시 **서버 세션 생성**
- `/check` API를 통해 **세션 유효성 검증**
- 로그아웃 시 **세션 제거 처리**

→ 세션을 활용한 **기본적인 사용자 인증 흐름 구현**

---

### 3. 비밀번호 보안 처리

- **SHA-256 + Salt 기반 비밀번호 해시 암호화 적용**
- 사용자별 **Salt 값을 저장하여 보안 강화**

→ 평문 비밀번호 저장을 방지하고 **보안성을 강화한 인증 시스템 구현**

---

### 4. 콘텐츠 목록 조회 및 필터 기능

- 콘텐츠 카테고리 **탭 기반 필터링 구현**
  - 시리즈
  - 영화
  - 애니메이션

- 콘텐츠 정렬 기능 구현
  - **평점순**
  - **제작 연도순**
  - **장르순**

---

### 5. 콘텐츠 상세 페이지

- 콘텐츠 상세 정보 표시
  - 줄거리
  - 평점
  - 제작 정보

- **트레일러 자동 재생 기능 구현**

---

### 6. REST API 구현

Spring Boot와 MyBatis를 활용하여 콘텐츠 및 회원 인증 관련 API를 구현했습니다.

| Method | Endpoint | Description |
|------|------|------|
| POST | `/signup` | 회원가입 |
| POST | `/login` | 로그인 |
| POST | `/logout` | 로그아웃 |
| GET | `/check` | 세션 유효성 확인 |
| GET | `/getContents` | 콘텐츠 목록 조회 |
| GET | `/contents/{id}` | 콘텐츠 상세 조회 |

---

## 🧠 프로젝트를 통해 얻은 것

- Spring Boot 기반 **웹 서비스 구조 이해**
- **세션 기반 인증 시스템 구현 경험**
- **SHA-256 + Salt 기반 비밀번호 암호화 처리 경험**
- MyBatis를 활용한 **데이터베이스 연동 및 API 구현 경험**
